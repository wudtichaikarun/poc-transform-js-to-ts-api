## Getting Start

---

```
# Get the latest snapshot
git clone https://github.com/wudtichaikarun/poc-transform-js-to-ts-api.git

# Change directory
cd poc-transform-js-to-ts-api

# Install NPM dependencies
npm install

# Start dev
npm run dev
```
